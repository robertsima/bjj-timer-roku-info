# Screenshots

Images in this folder are used for the Roku Channel Store listing and project README.

## Current files

- **01-setup-custom.jpg** — Custom preset: round length, break length, and total rounds
- **02-pre-round-starting.jpg** — Pre-round countdown (STARTING)
- **03-round-active.jpg** — Active round timer (ROUND)
- **04-open-mat-setup.jpg** — Open Mat preset with unlimited rounds
- **05-break.jpg** — Break countdown
- **06-session-complete.jpg** — Session summary (DONE)
- **collage-all-screens.jpg** — Overview collage of all screens

Roku typically accepts JPG or PNG store screenshots at 1280×720 (HD).
